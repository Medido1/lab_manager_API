# lab_manager_Backend

## Discription
  REST API backend for lab_manager, built with Node.js, Express, PostgreSQL, and Prisma.
  Handles authentication, client management, and file uploads.

## Tech Stack
 - Node.js
 - Expres
 - PostgreSQL
 - Prisma ORM
 - JWT Authentication
 - Multe

## Features
 - User authentication (register, login, JWT-based)
 - Client management (CRUD operations)
 - File upload & retrieval
 - RESTful API structure
